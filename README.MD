# Telegram VPN Bot с интеграцией 3X-UI и Multi-Node балансировкой

Асинхронный Telegram-бот на базе `aiogram 3.x` и `aiosqlite`, разработанный для развертывания подписок на базе панели **3X-UI (Xray)**. Бот поддерживает **FeatureToggle** для моментального переключения между бесплатным режимом выдачи подписок и режимом коммерческих продаж.

---

## Архитектура системы

```text
[Клиентские приложения] (V2Box, v2rayNG, NekoBox)
         │
         ▼ (Авто-апдейт конфигурации по ссылке подписки)
  [ Master-Сервер (Защищенный) ] 
  ├── Telegram-Бот (управляет логикой / БД SQLite)
  ├── Панель управления 3X-UI (Master-база)

         │
         ▼ (Синхронизация по API 3X-UI)
